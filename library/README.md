# safeslop library

Static assets used by the Go engine.

- `layer/container/` — canonical container assets (Dockerfiles, Squid allowlist,
  digest-pinned Squid OCI-index/manifests, per-package npm package-lock/SRI
  projects, and the agent-tools env example) copied into the Go embed tree by
  `make sync-container-assets` and verified by `make check`. Runtime Compose is
  generated by the Go engine; no standalone legacy Compose recipe is supported.
  Networked image builds are integrity-pinned but not hermetic or bit-reproducible,
  and only selected credential-free lock projects enter the Docker build context.

Everything the engine does at runtime is implemented by the `safeslop` Go binary —
use `safeslop validate`, `safeslop run`, `safeslop doctor`, and `safeslop down`.
The former fish/Python toolkit's policy fixtures, `.sb` Seatbelt profiles, and task
recipes were removed in specs/0054 (reference-only, superseded by the Go engine).
