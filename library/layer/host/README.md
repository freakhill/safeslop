# Host and sandbox layer

The host tier has no isolation boundary. The sandbox tier uses the macOS Seatbelt
profile generated by the Go engine to confine file and process access for
everyday agent work.

Use:

```bash
safeslop run <profile> --dry-run
safeslop run <profile>
```

with profiles such as:

```cue
profiles: review: {
	agent:       "claude"
	environment: "sandbox"
	network:     "deny"
}
```

For untrusted code or strict per-domain egress, prefer the container or VM tiers.
